# front-end-boilerplate

##Experimental Project

###Requirements
NodeJS
###Installation

````
$ npm install
````

### Run
````
$ gulp
````

### Avaliable tasks
- serve : Run browser sync in proxy mode. Access URL: 
`````
Local: http://localhost:3000
UI: http://localhost:3001
````

- sass : compile scss files: 
`````
Local: http://localhost:3000
UI: http://localhost:3001
````
